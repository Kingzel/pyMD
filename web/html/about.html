pyMD: Precision in Medical Diagnosis


pyMD is an innovative AI-powered web application meticulously designed to ask users a set of medical questions to arrive at a definite medical diagnosis.

pyMD was built and programmed in python using flask for the web deployment, and pandas, sklearn and numpy for the computing. The DDXplus simulated patient dataset was used to train pyMD and its classifier. It also uses a novel online policy based Reinforcement Learning approach to decide which question to ask next.
